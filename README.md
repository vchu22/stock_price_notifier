# Stock Price Notifier
A stock price desktop notifier built in Python. Compatible with MacOS, Linux, and Windows.

## Libraries used
- [Yahoo Finance](https://pypi.org/project/yahoo-fin/)
- [notify-py](https://pypi.org/project/notify-py/)
